# auto-install-openclaw

[OpenClaw](https://github.com/openclaw/openclaw)를 누구나 쉽게 설치할 수 있는 원클릭 GUI 인스톨러.

## 이게 뭔가요?

OpenClaw는 강력한 오픈소스 AI 어시스턴트 플랫폼이지만, 설치하려면 터미널 명령어, Node.js 설정, 수동 설정 파일 편집이 필요합니다. **auto-install-openclaw**는 이 모든 과정을 자동으로 처리하는 경량 GUI 인스톨러(~3MB)입니다.

## 사용 방법

1. `Setup.exe` 다운로드 & 실행
2. GitHub 계정으로 로그인 (OAuth)
3. GitHub Copilot 구독 상태 자동 확인
4. "설치" 클릭 — Node.js, OpenClaw, 설정 파일 모두 자동 설치
5. 완료 — 텔레그램, 디스코드, 웹챗으로 바로 연결

## 주요 기능

- 🚀 원클릭 설치 — 터미널 불필요
- 🔐 안전한 GitHub OAuth 인증
- ✅ Copilot 구독 자동 확인
- 📦 Node.js 없으면 자동 설치
- ⚙️ Claude Opus 4를 기본 LLM으로 자동 설정
- 🖥️ 시스템 시작프로그램 등록 (선택)
- 🪶 경량 스텁 인스톨러 (~3MB)

## 요구 사항

- Windows 10/11 (64비트)
- 인터넷 연결
- [Copilot 구독](https://github.com/features/copilot)이 활성화된 GitHub 계정
